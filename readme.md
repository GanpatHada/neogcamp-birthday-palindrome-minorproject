# IS YOUR BIRTHDAY IS PALINDROME

It is the web-app which checks the user birthday such that is birthday is palindrome or not <br>
if not,then tells the nearest palindrome date either in future or past

## TECHNOLOGY USED

HTML,CSS,JS
